# 만료일 계산기

매달 비용을 지불해야되는 유료 서비스가 있다.  
이 서비스는 다음 규칙에 따라 서비스 만료일을 결정한다.
- 서비스를 사용하려면 매달 1만원을 선불로 납부한다. 납부일 기준으로 한달 뒤가 서비스 만료일이 된다.
- 2개월 이상 요금을 납부할 수 있다.
- 10만원을 납부하면 서비스를 1년 제공한다.

### 테스트 기능명 정의
- 서비스 만료일을 계산하는 역할이므로 
  - 만료일은 'ExpiryDate'
  - 계산기는 'Calculator'

### 첫번째 테스트 
- 가장 쉽게 구현할수 있는 조건을 테스트로 정의
  - 만원 지불시 한달뒤가 만료일인 경우
